# Track Coach Pro ⏱️

**A cloud-synchronized, multi-user stopwatch application designed to give track and field coaching staffs real-time, multi-point timing and advanced pace analytics.**

🌐 **[Live App (Cloud Run)](https://track-coach-pro-775029592445.us-west1.run.app)**

## 📌 The Problem
Managing race times during a chaotic track practice or a crowded meet is incredibly difficult with standard timing apps. Coaches need to track multiple runners simultaneously, calculate pace splits on the fly, and compare current performance against personal bests. Furthermore, a single coach cannot be at the start line, the 200m mark, and the finish line at the same time, leading to fragmented and inaccurate data collection across the coaching staff.

## 🚀 The Solution
Built by a high school track coach and data analyst, **Track Coach Pro** is a scalable, staff-ready timing system. It allows multiple coaches stationed around the track to record splits and laps for the same active race in real time. It goes beyond basic timing by instantly calculating projected finish times and pace deltas, seamlessly exporting all data directly to Google Sheets for post-meet analysis.

### ✨ Key Features
* **Multi-Coach Synchronization:** Cloud-based architecture allows one coach to start the race and record laps at the finish line, while other coaches at different points on the track simultaneously record mid-race splits for the exact same event.
* **Advanced Pace Analytics:** Select specific race distances to automatically calculate real-time projected finish times and display exactly how far ahead or behind an athlete is compared to their target pace.
* **Roster & Goal Management:** Pre-load athlete rosters with customizable Personal Bests (PBs) or goal times for specific distances to benchmark active performance.
* **Google Sheets Integration:** Export complete race logs, lap splits, and pace data directly to a Google Sheet with a single click for permanent record-keeping and long-term athlete development tracking.

## 💻 Technical Stack
* Tech Stack: TypeScript, Node.js, Firebase, Vite
* **Core Logic & Development:** Google AI Studio
* **Deployment:** Google Cloud Run
* **Data Export:** Google Sheets API
